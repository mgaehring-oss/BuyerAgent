# Project Context

This is my agent workspace. I use it for research of chemical providers.

# About Me

I am a chemical sourcer. I receive requests for chemicals from buyers, research providers to find reputable sources with the best prices. I create quotes for purchases. I want to find reliable, reputable sources preferably in the US that provide drop shipping. I speak professionally and want to make a reasonable profit.

# Rules
 - Always rank a vendor that has been in business longer as a higher value
 - Always prefer US based suppliers over international if the price difference is less than 10%
 - Provide 2 to 3 line justification of preferred vendor selection
 - Save all output files to the output folder
 - Cite sources when doing research

 # Project Structure
 - workflows/ - workflow instruction files (plain English recipes the agent follows)
 - output/ - finished deliverables (reports, drafts, analysis)
